# Action Framework Skill

This skill contains the Action development methodology and its practical implementation through action-lite.

## Contents

- **SKILL.md** - Main skill file containing the complete Action framework methodology and action-lite file-based implementation
- **references/** - Supporting documentation and protocol specifications

## Purpose

The Action framework ensures that design and testing phases are built into the development process and cannot be skipped. Action-lite provides a file-based task tracking system using markdown files and meta-graphs for dependency management.

## When to Use

Use this skill when:
- Planning development work
- Creating specifications for features
- Setting up task tracking systems
- Needing structured design before implementation
- Managing complex task dependencies
